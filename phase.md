# [Project Asset] AI 시대의 웹 제작 마스터 — 20시간 완전 자동화 커리큘럼

## 1. 상세 강의 로드맵 (Total 20 Hours / 25 Chapters)

### Phase 1. 웹의 뼈대와 근육 (HTML & CSS 기초) — 4시간
* **01강:** 웹사이트는 어떻게 작동하는가? (도메인, 호스팅, 브라우저)
* **02강:** HTML5: 태그로 만드는 웹의 골격 (제목, 본문, 이미지, 링크)
* **03강:** CSS3 기초: 색상, 폰트, 그리고 박스 모델(Box Model) 이해하기
* **04강:** 레이아웃의 정석: Flexbox와 Grid로 배치 끝내기
* **05강:** [실습] 나만의 간단한 자기소개 페이지 만들기

### Phase 2. AI로 10배 빠르게 제작하기 (AI 툴 마스터) — 4시간
* **06강:** v0.dev & Bolt.new: 프롬프트 한 줄로 웹 UI 생성하기
* **07강:** ChatGPT & Claude를 활용한 코드 디버깅 및 기능 추가
* **08강:** Cursor 에디터 입문: AI와 함께 코딩하는 혁신적인 방법
* **09강:** AI 이미지 생성(Midjourney/DALL-E)으로 웹 에셋 만들기
* **10강:** [실습] AI를 이용해 1시간 만에 랜딩 페이지 초안 완성하기

### Phase 3. 세련된 디자인과 반응형 웹 (Tailwind CSS) — 4시간
* **11강:** Tailwind CSS: 유틸리티 우선 방식의 혁명적 디자인법
* **12강:** 반응형 디자인: 모바일, 태블릿, PC 대응 전략
* **13강:** 다크 모드 및 애니메이션 효과로 생동감 불어넣기
* **14강:** UI/UX 원칙: 비전공자가 꼭 알아야 할 디자인 기본기
* **15강:** [실습] 기업형 반응형 랜딩 페이지 고도화 작업

### Phase 4. 동적인 웹사이트와 기능 확장 (JavaScript & API) — 4시간
* **16강:** 자바스크립트 기초: 변수, 함수, 그리고 이벤트 처리
* **17강:** 외부 라이브러리 활용: 스크롤 애니메이션, 슬라이더 적용
* **18강:** AI 챗봇(OpenAI API) 내 웹사이트에 이식하기
* **19강:** 구글 폼 & 이메일 전송 기능 연동하기
* **20강:** [실습] 문의하기 기능이 포함된 실전 비즈니스 사이트 제작

### Phase 5. 세상에 내놓기 (배포, SEO, 운영) — 4시간
* **21강:** GitHub 기초: 내 코드를 안전하게 관리하는 법
* **22강:** Vercel & Netlify를 이용한 1분 초고속 무료 배포
* **23강:** 도메인 연결 및 HTTPS 보안 설정하기
* **24강:** 검색 엔진 최적화(SEO): 구글 검색 결과 상단 노출 전략
* **25강:** 구글 애널리틱스(GA4)로 방문자 분석 시작하기
* **최종:** 나만의 포트폴리오 사이트 최종 완성 및 검수
